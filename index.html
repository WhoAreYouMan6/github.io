Installation & Usage

Requirements

  • Minecraft 1.21
  • Fabric Loader 0.15.11+
  • Meteor Client 0.5.8+
  • Java 21+

Installation

  1. Download the JAR file
  2. Place in your Minecraft mods folder
  3. Launch Minecraft with Fabric
  4. Open Meteor Client ClickGUI
  5. Find "Egli's Autoplayaddon-addon" in modules
